# Development
## Prerequisites
You will need:
- Node v8 or Newer

## Installation
To install simply run `yarn` or `npm install`

## Development
To develop locally run `yarn start` or `npm run start`


## Deployments
To deploy run `yarn deploy` or `npm run deploy`
